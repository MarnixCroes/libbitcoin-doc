# Libbitcoin Docs

Documentation for Libbitcoin. Built using mkdocs material theme.

### Install
```
sudo apt-get install mkdocs
sudo apt-get install mkdocs-material
```

### Build
```
mkdocs build
```

### Run
```
mkdocs serve
```
